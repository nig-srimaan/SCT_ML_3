# Cat vs Dog Image Classification using SVM

A machine learning pipeline that trains a Support Vector Machine (SVM) to classify images of cats and dogs. The model is built using OpenCV for image preprocessing and scikit-learn for the classification algorithm.

## Dataset

This project uses the standard Microsoft Cats and Dogs dataset. Because the dataset is over 800MB, it is not included in this repository. 

You can download the dataset here: [Kaggle: Cats and Dogs Dataset](https://www.kaggle.com/datasets/tongpython/cat-and-dog) (or your preferred source).
